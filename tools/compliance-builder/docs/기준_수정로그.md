## 미해결/노트

기존 `파싱오류_수정결과.txt` (별표5 2건, 별표5의3 5건, 별표7 7건, 별표9 3건, 별표11 3건)는
이미 모두 적용되어 있는 것을 확인함.

추가로 끊김으로 의심되는 항목은 발견하지 못함. 자동 탐지(`flag-truncated-text.js`)로
202건 후보가 나왔으나 모두 다음 중 하나의 정상 케이스로 확인됨:
- 정상 종결("…아니하다", "…한다", "…경우" 등)이 종결규칙 사전에 누락된 경우
- 섹션 헤더 항목(isHeader=true) — 짧은 명사형 종결(예: "기계적 처분시설")
- 명사형 항목/표 항목 — 정상적으로 명사로 끝남(예: "보관창고", "냉장시설")
