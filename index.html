이 저장소는 서로 성격이 다른 HTML 툴들을 한곳에 모아 배포하기 위한 시작점입니다. 앞으로 툴을 만들 때마다 폴더 하나씩 추가하고, GitHub에 push하면 홈페이지에 바로 반영되게 구성하면 됩니다.

현재 등록된 툴

새 툴 추가 방식

1. 폴더 생성

tools/새-툴-이름/index.html 형태로 추가하면 됩니다.

2. 메타 정보는 선택

필요하면 각 툴 HTML의 meta 태그로 이름, 설명, 분류를 넣고, 없어도 기본값으로 자동 등록됩니다.

3. GitHub에 push

main 브랜치에 push하면 GitHub Actions가 목록을 만들고 GitHub Pages에 자동 배포합니다.
