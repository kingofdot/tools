새 툴 추가 방식

1. 폴더 생성

tools/새-툴-이름/index.html 형태로 추가하면 됩니다.

2. 홈에 카드 추가

이 페이지의 카드 한 칸만 복사해서 이름, 설명, 링크만 바꾸면 모음집에 노출됩니다.

3. GitHub에 push

main 브랜치에 push하면 GitHub Pages 배포 워크플로가 자동 실행됩니다.
